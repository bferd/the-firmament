# Videos Directory

Place your character video files here:

- `hero-welcome.webm` — plays once on load
- `hero-idle-loop.webm` — loops on hero
- `hero-transition.webm` — plays on scroll trigger
- `hero-browse-idle.webm` — loops in side panel
- `hero-background.mp4` — hero background video

Videos are not included in this repository.
Generate your own using Higgsfield.ai or 
similar AI video generation tools.
